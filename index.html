Drawing Lines in the Sand

Now that we understand how a perceptron makes decisions, let's visualize what it's actually doing. Imagine you're at the beach, and you're trying to separate seashells from rocks by drawing a line in the sand.

Continue

This is exactly what a perceptron does! Let's say we measure each object's size (x₁) and smoothness (x₂). The perceptron draws a dividing line in this two-dimensional space. Everything on one side it calls 'seashell', everything on the other side 'rock'.

The equation of this line is beautifully simple:

\[ w₁x₁ + w₂x₂ + b = 0 \]

The weights (w₁, w₂) determine the line's slope, and the bias (b) shifts it up or down.

Decision Boundary

The line (or plane in higher dimensions) that separates different classes in the input space. For a perceptron, this boundary is always linear.
